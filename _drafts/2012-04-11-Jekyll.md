### Liquid Syntax

- include parameters

使用`{% include %}`是可以带参数，参数可以传递对应的`HTML`文件

```javascript
<footer>
{% include nav.html class='nav-bottom' %}
</footer>

//in nav.html
<nav>
    <ul class="nav {{include.class}}">
    ...
    </ul>
</nav>
```

### Jekyll Tips

- looping over posts

```javascript
<ul>
  {% for post in site.posts %}
    <li>
      <a href="{{ post.url }}">{{ post.title }}</a>
    </li>
  {% endfor %}
</ul>
```
