### Rust Overview

- A New Programming Lanuage
- Made by the Mozilla Corporation
- Already had 1.0 release
- Native Code
- Powerful and safe
- An alternative C/C++, D, Go=o, ...
- Introduces new and insteresting concepts

### Install Rust

- macOS
    - `brew install Rust`
- Compile Rust
    - `rustc main.rs`
- Package Manager
    - `cargo`
